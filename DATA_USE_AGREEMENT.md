# PolliFlower Data Use Agreement

This Data Use Agreement governs access to and use of the complete PolliFlower dataset.

The complete PolliFlower dataset is provided for academic research and educational purposes only. By requesting, downloading, accessing, or using the dataset, the user agrees to the following terms.

## 1. Permitted Use

The dataset may be used only for non-commercial academic research, education, and benchmarking.

Permitted uses include:

- training and evaluating computer vision models;
- studying precision pollination perception;
- developing detection, segmentation, or keypoint localization methods;
- reproducing or extending the PolliFlower benchmark.

## 2. Prohibited Use

The user may not:

- redistribute, sublicense, sell, lease, publish, or publicly host the full dataset;
- upload the full dataset or substantial subsets to public platforms, including but not limited to GitHub, Kaggle, Hugging Face, Google Drive public links, or cloud storage with public access;
- use the dataset for commercial products, commercial services, or commercial model training without explicit written permission;
- use the dataset for unlawful, harmful, or unethical applications;
- remove, modify, or obscure attribution, ownership, or license information;
- share the dataset with individuals or organizations not covered by this agreement.

## 3. Data Security

The user agrees to store the dataset securely and take reasonable measures to prevent unauthorized access, copying, or redistribution.

If the dataset is stored on institutional servers, cloud platforms, or shared computing clusters, access must be restricted to authorized project members only.

## 4. Publication and Citation

Any publication, report, presentation, preprint, or derivative work using the PolliFlower dataset must cite the PolliFlower paper after publication.

The citation entry will be provided after the paper is published.

## 5. Derived Data and Models

Users may publish research results, trained model performance, and derived analysis based on the dataset.

However, users may not publish or redistribute derived files that allow reconstruction of the original dataset, including complete annotations, complete image subsets, or converted full-dataset copies.

Trained model weights may be released only if they do not contain or reconstruct the original images or annotations and if such release does not violate any institutional or third-party restrictions.

## 6. Commercial Use

Commercial use is not permitted under this agreement.

Any commercial use, including use in commercial products, services, paid model training, or industrial deployment, requires prior written permission from the PolliFlower maintainers.
